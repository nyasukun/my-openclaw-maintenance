# Learning KB

Use this skill for learning and knowledge-base work.

Organize material into:

- source or topic
- summary
- key concepts
- links to related notes
- review questions
- follow-up study tasks

Prefer reusable notes over one-off summaries. Preserve citations or source
references when provided.
