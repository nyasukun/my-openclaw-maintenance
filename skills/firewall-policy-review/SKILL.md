# Firewall Policy Review

Use this skill for firewall and segmentation policy review.

Check:

- source and destination scope
- service and port necessity
- rule order and shadowing
- logging and monitoring
- expiry or change ticket
- least-privilege alternatives

Flag missing context instead of assuming it.
