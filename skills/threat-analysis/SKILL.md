# Threat Analysis

Use this skill for threat modeling and security analysis.

Organize findings by:

- asset or process at risk
- threat scenario
- impact
- likelihood
- existing controls
- detection
- mitigation priority

Avoid overstating certainty when evidence is incomplete.
